# Import TAH IDE Settings
this project make it easy to import and export IDE settings
- tah is my nickname
- this project for personal use

## Feature
- import settings, extensions, and keybindngs from `TAHPAPANGKORN/tahaiiya-ide-settings` repo

## Usage

- ### import
```bash
npx TAHPAPANGKORN/tahaiiya-ide-settings 
```

- ### export
```bash
npx TAHPAPANGKORN/tahaiiya-ide-settings export
```
